## Build Philosophy

1. **Each phase ships something usable.** No phase leaves the app broken.
2. **Prove the card quality before scaling.** Phase 0 builds the pipeline, locks a reference figure, and generates + **approves** the Strength A cards before the full library.
3. **Two phases, kept separate.** Author-time generation (online, occasional) never mixes with the offline training runtime.
4. **Offline runtime is non-negotiable.** `trainer.html` makes zero network calls; the API key lives only in `.env`.
5. **Nothing un-approved ships.** Every card passes a human approval gate before the app uses it.
6. **A full session first.** Generation is ordered **by session** (Strength A → Strength B → Warm-up → Mobility) so a complete, trainable session is ready soonest — that's the magic moment.
7. **Evidence must survive.** Export/import ships in the core phase.
8. **Strictly the knowledge base.** Only exercises Ivan authored — nothing invented. The plan is a data object (Foundation Plan v1) so a future plan is a data add, not a rebuild; the KB 8-week ramp drives a "this week's target" that eases a returning trainee in.

> Delivery is a small folder — `trainer.html` + `images/` — still offline and server-free. Generation uses the OpenAI **platform API** (pay-as-you-go, not ChatGPT Plus). No GitHub repo required; per-phase PR review optional.

### Foundation Plan v1 — sessions (strictly from the knowledge base; nothing invented)
The Strength A/B split is **derived from the KB's stated purposes**, matching each library exercise to A (push/pull/squat/core) or B (shoulders/back/legs/core). Adjustable when authoring data.
- **Strength A** (5): Band Chest Press, Band Row, Bodyweight Squat, Dead Bug, Front Plank.
- **Strength B** (6): Band Overhead Press, Band Face Pull, Band Lat Pulldown, Reverse Lunge, Band Romanian Deadlift, Side Plank.
- **Warm-up** (8): Neck mobility, Shoulder circles, Arm swings, Thoracic rotation, Hip circles, Cat–Cow, Bodyweight Squat (shared with Strength A), March in place.
- **Mobility** (8): Doorway chest opener, World's Greatest Stretch, Hip flexor stretch, Hamstring stretch, Calf stretch, Shoulder wall slides, Child's pose, Supported deep squat hold.
- **Progression** (KB 8-week ramp): wk 1–2 → 2 sets, light band; wk 3–4 → 3 sets; wk 5–6 → increase resistance if stable; wk 7–8 → toward 12–15 reps.
- ~26 unique cards (Bodyweight Squat shared). Bird-Dog is **not** in the KB and is excluded.
